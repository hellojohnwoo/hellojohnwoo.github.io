Diary-2020-06-20

Posted on 2020-06-20 In Diary

最近悟った考え

  • 私は強く言いたい。 すべての人はプログラミングを学ばなければならないと思う。
  • 結論：
    • プログラミングを学ぶことで、自分の性格を知ることができ、
    • どのような人々と働きたいか、
    • そしてそのようなグループの中で、どのようなシナジー的な成果が達成できるかも分かるようだ。

過去の私

  • 私はJava, Pythonが私のプログラミングランゲージだった.
  • 理由としては、Javaのシェアが高く、ローレベルランゲージであったことから、大規模なSI環境でも高い完成度と信頼性のランゲージであったからだと思う。
  • 特に、Spring framework はMVC モデリングの頂点であったと考える。

現在の私

  • 近年では、JavaScript、Pythonの目覚ましい発展と強力なライブラリにより、
    ローレベル言語に匹敵するレベルになっていると考えている。
  • JavaScript : React + Node + AWS(Azure, GCP)
  • Python : Django API
  • すごく面白くて面白い。久しぶりに感じるプログラミングに対する熱情。
  • スクリプトランゲージの変化が早すぎるため、過去とは本当に変わったといってもおかしくない。

未来の私

  • JavaScript、Pythonフルスタックエンジニアとして働きたい。

  • 既に新入レベルのJavaScriptフルスタック開発者として競争力があると思う。 残念な点があるとしたら、フロントエンドなデザイン感覚がまだ足りない。 コンポーネントとデザイン的な試行錯誤がちょっと必要だ。
    ｄｉｖとの戦いが再び始まるかも知れない。

  • PythonではDjango-Server APIの開発が可能である。
    Flaskにするアーキテクチャの勉強をもっとしなければならない。 完成度がまだ低い。
